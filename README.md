# StickyNav
StickyNav is a Vanilla JS lightweight Sticky Navbar script using animate.css

## Basic Usage
```javascript
bower install
```

And then..
```html
<script src="stickynav.js"></script>

<script type="text/javascript">
	stickyNav.init();
</script>
```
